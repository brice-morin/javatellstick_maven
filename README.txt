THIS PROJECT IS EASY TO IMPORT TO ECLIPSE!

---------Instructions-----------


1. Download eclipse.

2. Setup workspace folder in eclipse.


3. Create a dir named TellstickDUO inside workspace.

4. Open eclipse, go to:
 File -> New -> Java Project.

5. As project name: type in: "TellstickDUO" , it will now find all the sources and project info.

6. Click finish!

7. If you don't got 32 bit Java, you will need to install it and add it to eclipse.
   You can also easily add 64 bit dll to lib/ folder and then you can run it on 64bit java.
